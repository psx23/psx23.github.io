PlayStation®3 4.90 HEN Установщик



Внимание!!! На PS3 должна быть установлена гибридная прошивка (HFW) версии 4.90.

Если установлен WebMan, то обновите его до последней версии перед установкой HEN.

Инструкция по установке HEN:

Нажмите СКАЧАТЬ ФАЙЛ 1, далее нажмите СКАЧАТЬ ФАЙЛ 2, далее нажмите УСТАНОВИТЬ HEN. Браузер должен закрыть страницу автоматически и если предложит закрыть страницу, нажмите OK и немного подождите, должен установиться файлик. После установки файла перезарузите PS3. HEN установлен.
